GSAP: Método set( )

É um método que seta as configurações iniciais do elemento.

gsap.set(elemento, {x: 0, y: 0, skewX:"-30deg",});

GSAP: Método to( )

É um método que seta as configurações Finais do elemento.

gsap.to(elemento, .5, {y: 100, rotateY: 180,});

GSAP: Método from( )

É um método que considera as partes de um estado inicial e o CSS como o estado final do elemento.

gsap.from(elemento, {scaleY: 0.2, scaleX: 0.3, opacity: 0.1, duration: 1.5, });

GSAP: Método fromTo( )

Em um mesmo método define o estado inicial e o final do elemento.

gsap.fromTo(elemento,{ rotateX: "-180deg", },{ rotateX: "980deg", duration: 2.5, ease: "circ.inOut", });

GSAP: Método stagger( )

Método similar ao to(), from() e fromTo(), só que tem manipulação de tempo por delays e é aplicado em listas.
*'OBS. Descontinuado no v3 porisso utilizar TweenMax'

TweenMax.staggerFromTo(listaElementos, 2, { scale: 0, rotate: "0deg", }, { scale: 1, rotate: "360deg" }, 0.2)

GSAP: Loops

Um loop permite que uma animação se repita um número específico de vezes ou indefinidamente.

gsap.fromTo(elemento,{ rotate: "0deg", y: 0, x: 0, }, { rotate: "360deg", y: 100, x: 100%, duration: 2.5, boxShadow: "0 0 32px", ease: "elastic.inOut(1,0.3)", delay: index * 0.3, yoyo: true, repeat: -1, });
